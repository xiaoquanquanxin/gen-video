# Seedance 视频生成应用 - 运行指南

## 首次运行（从零开始）

### 1. 安装 Python 依赖

```bash
cd backend
pip install -r requirements.txt
```

### 2. 安装 Node.js 依赖

```bash
cd frontend
npm install
```

### 3. 配置 API Key

编辑 `backend/.env` 文件，填入你的火山引擎方舟 API Key：

```
ARK_API_KEY=你的真实API_Key
```

### 4. 启动后端服务

```bash
cd backend
uvicorn main:app --reload --port 8000
```

### 5. 启动前端服务（新开一个终端）

```bash
cd frontend
npm run dev
```

### 6. 访问应用

浏览器打开 http://localhost:3000

---

## 第 N 次运行（已安装过依赖）

只需启动两个服务即可：

**终端 1 - 后端：**
```bash
cd backend
uvicorn main:app --reload --port 8000
```

**终端 2 - 前端：**
```bash
cd frontend
npm run dev
```

---

## 快捷命令（Windows PowerShell）

如果你想一行命令启动，可以用：

```powershell
# 后端
Start-Process powershell -ArgumentList "-NoExit", "-Command", "cd backend; uvicorn main:app --reload --port 8000"

# 前端
Start-Process powershell -ArgumentList "-NoExit", "-Command", "cd frontend; npm run dev"
```

## 常见问题

### Q: 端口被占用怎么办？

后端换端口：
```bash
uvicorn main:app --reload --port 8001
```
同时修改 `frontend/.env.local`：
```
NEXT_PUBLIC_API_URL=http://localhost:8001
```

### Q: 如何停止服务？

在对应终端按 `Ctrl + C`

### Q: 依赖更新后怎么办？

```bash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd frontend
npm install
```
